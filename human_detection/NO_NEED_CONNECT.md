# 无需预先连接机械臂

## 工作原理

**你不需要先连接机械臂！**

系统采用**自动连接**设计：

```
检测到人
    ↓
自动调用 wave_hand_standalone.py
    ↓
脚本内部执行:
  1. 连接机械臂
  2. 执行挥手
  3. 回到原点
  4. 断开机械臂
```

## 与旧版本的区别

| 旧版本 | 新版本 |
|--------|--------|
| Web GUI 先连接机械臂 | Web GUI 不连接机械臂 |
| 保持长连接 | 按需连接，用完即断 |
| 可能冲突导致崩溃 | 完全隔离，稳定可靠 |

## 使用步骤

### 1. 启动 Web GUI
```bash
cd /home/aidlux/human_detection
python3 web_gui.py
```

### 2. 点击"开始监控"
在网页上点击绿色的"开始监控"按钮。

### 3. 走到摄像头前
- 让人站在摄像头前
- 距离 2 米以内
- 等待系统检测

### 4. 自动触发
检测到人后，系统会：
1. 自动连接机械臂
2. 执行挥手动作
3. 回到初始位姿
4. 自动断开连接

## 配置检查

确保 `web_gui.py` 中的配置正确：

```python
config = {
    "enable_wave_action": True,   # 必须 True
    "wave_interval": 60,          # 触发间隔(秒)
    "welcome_distance": 2.0,      # 迎宾距离(米)
}
```

## 日志查看

启动 Web GUI 后，观察日志输出：

```
[挥手] 检测到人，距离: 1.5m，可触发: True
[挥手] 已触发 (距离: 1.5m)
[SimpleWave] 执行挥手动作...
```

如果没有看到 `[挥手] 检测到人` 的日志，说明：
1. 摄像头没有检测到人
2. 人体检测器未正常工作
3. 距离超过了 `welcome_distance`

## 常见问题

### Q: 为什么 Web GUI 里没有"连接机械臂"按钮？
**A**: 这是正常设计。机械臂连接在挥手时自动完成，无需手动连接。

### Q: 如何知道机械臂是否已连接？
**A**: 查看日志输出 `[1/6] 连接机械臂 192.168.3.100...` 和 `[1/6] ✓ 连接成功`。

### Q: 检测到人但日志显示"冷却中"？
**A**: 触发间隔保护，默认 60 秒内不会重复触发。可以修改 `wave_interval` 缩短间隔。

### Q: 可以手动触发挥手吗？
**A**: 可以。在 Web GUI 的"挥手动作"面板中点击"触发挥手"按钮。
