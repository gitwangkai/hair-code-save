# 挥手动作完整指南

## 动作流程

当检测到人时，系统会执行以下完整动作：

```
检测到人
    ↓
触发动作
    ↓
┌─────────────────────────────────────┐
│  1. 回到原点 (HOME)                  │
│     所有关节归零，确保起始状态一致    │
│     延时: 1.5秒                       │
├─────────────────────────────────────┤
│  2. 移动到准备姿势                    │
│     举起手臂，准备挥手               │
│     延时: 1.5秒                       │
├─────────────────────────────────────┤
│  3. 执行挥手动作 (3次)               │
│     左摆 → 右摆 → 左摆 → 右摆...    │
│     每次延时: 0.3秒                   │
├─────────────────────────────────────┤
│  4. 回到准备姿势                      │
│     恢复中间位置                     │
│     延时: 0.5秒                       │
├─────────────────────────────────────┤
│  5. 回到原点 (HOME)                  │
│     必须执行，确保安全               │
│     延时: 1.5秒                       │
├─────────────────────────────────────┤
│  6. 伺服下电                          │
│     关闭电机，进入待机状态           │
└─────────────────────────────────────┘
    ↓
播放音频 (与挥手并行执行)
    ↓
动作完成
```

## 安全措施

### 自动恢复机制
- **正常完成**: 步骤 5 自动回到原点
- **执行出错**: `finally` 块确保回到原点
- **进程崩溃**: `atexit` 注册紧急恢复
- **用户中断**: 捕获 Ctrl+C 执行恢复

### 关节限位保护
每个动作前都检查关节角度是否在安全范围内：
- J1: -70° ~ 200°
- J2: 0° ~ 120°
- J3: -55° ~ 55°
- J4: 0° ~ 120°
- J5: -85° ~ 85°
- J6: -20° ~ 20°

## 关键姿势角度

### 原点 (HOME)
```
[J1, J2, J3, J4, J5, J6] = [0, 0, 0, 0, 0, 0]
```
所有关节归零，机械臂呈自然下垂状态

### 准备姿势
```
[J1, J2, J3, J4, J5, J6] = [64.3, 0, 13.2, 85.9, -84.8, 0]
```
手臂举起，准备挥手的位置

### 挥手摆动
- **左摆**: J3 = 20° (向前)
- **右摆**: J3 = -20° (向后)

## 配置参数

### 修改触发间隔
在 Web GUI 的"挥手动作"面板中，或修改配置：
```python
"wave_interval": 60  # 秒，默认60秒
```

### 修改挥手次数
编辑 `src/wave_hand_safe.py`：
```python
WAVE_TIMES = 3  # 挥手次数，默认3次
```

### 修改摆动幅度
编辑 `src/wave_hand_safe.py`：
```python
WAVE_LEFT = [64.3, 0, 20.0, 85.9, -84.8, 0, 0]   # 左摆幅度
WAVE_RIGHT = [64.3, 0, -20.0, 85.9, -84.8, 0, 0] # 右摆幅度
```

## 故障排除

### 问题: 动作完成后没有回到原点
**原因**: 可能是机械臂通信中断
**解决**: 手动执行回原点：
```bash
cd /home/aidlux/demo_arm
python3 -c "
from PallasSDK import Controller, LocationJ
ctrl = Controller()
ctrl.Connect('192.168.3.100')
ctrl.SetPowerEnable(True)
robot = ctrl.AddRobot(1)
robot.SetFrameType(1)
robot.MoveJ(LocationJ(0,0,0,0,0,0))  # 回原点
"
```

### 问题: 某个关节超限
**现象**: 日志显示 "J3=XX.X° 超限"
**解决**: 修改 `Config.WAVE_LEFT` 或 `Config.WAVE_RIGHT` 中对应关节的角度值

### 问题: 动作执行太慢
**原因**: 延时设置过长
**解决**: 修改 `wave_hand_safe.py` 中的延时参数：
```python
delay=1.0  # 减小延时值
```

## 注意事项

1. **触发间隔**: 建议不小于 30 秒，给机械臂足够的休息时间
2. **原点位置**: 必须确保机械臂在原点时不会与周围物体碰撞
3. **电源管理**: 动作完成后会自动下电，需要时重新上电
4. **紧急停止**: 如需立即停止，在 Web GUI 中点击"停止动作"或按 Ctrl+C
