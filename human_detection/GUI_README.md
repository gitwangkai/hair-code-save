# 🖥️ 智能迎宾机器人 Web GUI

基于 Flask + WebSocket 的实时监控系统，可以在浏览器中查看检测效果。

## 📋 功能特性

- ✅ **实时视频流** - 通过浏览器查看摄像头画面
- ✅ **检测可视化** - 实时显示人体检测框和人脸识别结果
- ✅ **系统状态监控** - FPS、检测人数、云台角度等
- ✅ **远程控制** - 启动/停止监控、截图、注册人脸
- ✅ **日志显示** - 实时查看系统运行日志
- ✅ **迎宾提示** - 检测到访客时显示欢迎信息

## 🚀 快速启动

### 方法一：使用启动脚本

```bash
cd /home/aidlux/human_detection
./start_gui.sh
```

### 方法二：手动启动

```bash
cd /home/aidlux/human_detection
python3 web_gui.py
```

## 🌐 访问地址

启动成功后，在浏览器中打开显示的地址：

```
http://<IP地址>:<端口>
```

例如：`http://192.168.0.110:5395`

- **总访客** - 检测到的总访客数

- **未知访客** - 未注册的访客数
- **云台角度** - 当前云台 Pan/Tilt 角度

## 🔧 高级功能

### 注册人脸

1. 确保系统正在运行
2. 让人面对摄像头
3. 点击 "👤 注册人脸" 按钮
4. 输入姓名并确认
5. 等待提示 "注册成功"

### 截图保存

点击 "📷 截图" 按钮，当前画面将自动下载到本地。

### 日志查看

右侧面板显示系统运行日志，包括：
- 系统启动/停止
- 人体检测结果
- 人脸识别结果
- 迎宾触发记录
- 错误信息

## ⚙️ 配置说明

可以在 `web_gui.py` 中修改以下配置：

```python
config = {
    "camera_source": "/dev/video_header",  # 摄像头设备
    "width": 640,                          # 画面宽度
    "height": 360,                         # 画面高度
    "welcome_distance": 2.0,               # 迎宾距离(米)
    "enable_ptz": True,                    # 启用云台追踪
}
```

## 🐛 常见问题

### Q: 无法访问页面?
A: 检查防火墙设置，确保端口开放。可以尝试使用 `localhost` 访问。

### Q: 视频流卡顿?
A: 降低分辨率或帧率，或检查网络带宽。

### Q: 检测效果不准确?
A: 调整 `confidence_threshold` 参数，或改善光线条件。

## 📝 技术栈

- **后端**: Flask + Flask-SocketIO
- **前端**: HTML5 + JavaScript + WebSocket
- **视频流**: MJPEG over WebSocket
- **样式**: CSS3 (深色主题)
